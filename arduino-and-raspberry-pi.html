Arduino and Raspberry Pi

Permalink: 2014-01-04 00:00:00-06:00 by Keith Gray in personal tags: arduino

My father, brother, and I are starting to have weekly or bi-weekly sessions so I can teach them about the arduino and raspberry pi. It will also give us a scheduled time to work on a project. Finally, it will be good time spent together for the three of us.

Session 1 Agenda and Links

  1. Introduction to the Arduino boards and software

    • Uno
    • Leonardo
    • Others

  2. Install the Arduino Software in Windows or Linux

  3. Connect the Arduino to the PC and make sure the software can communicate with it

    1. Choose the Leonardo from the list of boards
    2. Choose the correct serial port
    3. Choose the blink example from the menus, compile it, and send it to the Arduino
    4. Make sure there are no errors
    5. Make sure the onboard LED connected to pin 13 is blinking

  4. Visually insped the code and describe how it works

    • Declaring constants (different than what is in the examples)
    • setup function
    • loop function
    • Built in function: pinMode
    • Built in function: digitalWrite
    • Built in function: delay

  5. Use an external LED

    1. Wire up and external LED with the correct resistor value
    2. Alter the Arduino code to choose the correct pin
    3. Compile and send to the Arduino
    4. Verify the LED blinks

  6. Play with the code a little bit

    1. Make the LED stay on longer
    2. Make the LED stay off longer
    3. Make the LED blink faster with equal on/off times (duty cycle)

  7. Talk about Pulse Width Modulation (PWM) (A form of analog output)

    • Built in function: analogWrite

  8. Use PWM to fade an LED

    1. Open the Fade example from the Arduino IDE menu
    2. Inspect the code to see what it does
    3. Set the led pin correctly in the code for how the LED is connected
    4. Compile and send the code to the Arduino
    5. Verify the LED fades in and out

  9. Play with the code a little bit

    1. Make it take longer to fade in and out
    2. Make it fade faster
    3. Make it so it only fades out and then goes immediately to full brightness again
